Web Page Canvas v2.0.0

This is a free-to-use extension that requires less than 3MB of storage and allows you to draw on your current web page.

The extension does not work on pdf files, images and on chrome's default pages.

Hello dear Google Chrome user! My name is Damianos Giankakis, and I am the developer of this extension.

This extension started as a hobby, and helped me improve my programming skills.
